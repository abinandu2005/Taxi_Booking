# 🚖 Taxi Booking System (Java Console Application)

## 📌 Project Overview

The **Taxi Booking System** is a simple console-based Java application that simulates a taxi booking service.
It allows users to book taxis, assign the nearest available taxi, and manage bookings efficiently.

This project demonstrates basic **Java programming concepts such as OOP, classes, packages, and logic building.**

---

## 🛠 Technologies Used

* Java
* Object Oriented Programming (OOP)
* Console-based Interface

## ⚙️ Features

* Book a taxi
* Automatically assign available taxi
* Calculate fare based on distance
* Track taxi booking details
* Simple console interaction

## ▶️ How to Run the Project

1. Clone the repository

```
git clone https://github.com/yourusername/TaxiBooking.git
```

2. Open the project in **Eclipse / IntelliJ / VS Code**

3. Navigate to:

```
src/com/taxibooking/main/Main.java
```

4. Run the **Main.java** file.
